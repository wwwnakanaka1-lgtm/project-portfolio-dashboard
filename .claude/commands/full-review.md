# Full Repository Review (Manual)

1) Generate bundle:
   scripts/repo-review-bundle.ps1

2) Open docs/REPO_REVIEW_BUNDLE.md and paste it into ChatGPT for a full review.

Options:
- Max file size: scripts/repo-review-bundle.ps1 -MaxFileSizeKb 200
- Max total size: scripts/repo-review-bundle.ps1 -MaxTotalKb 800
- Include docs: scripts/repo-review-bundle.ps1 -IncludeDocs
